# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please report security issues via [GitHub Issues](../../issues)** with the "security" label.

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This project is a Claude Code extension that generates code and configuration files. Security concerns may include:

- Injection of malicious content into generated files
- Exposure of sensitive data from project briefs or configuration
- Unsafe shell command execution in scripts
